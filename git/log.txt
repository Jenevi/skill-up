30.07.25
- HEAD - тот коммит, над которым сейчас работаем
- git checkout [hash] - позволяет установить HEAD на нужный коммит
- HEAD^ - на один коммит ближе к исходному
- HEAD~4 - на четыре коммита ближе к исходному
- git branch -f main HEAD~3 - принудительно переместить ветку main на три родителя назад от HEAD
- git cherry-pick [hash] - в текущую ветку встраивает коммиты [hash]
- git rebase -i HEAD~2 - позволяет интерактивно перетасовать ветку
- git commit --amend - позволяет внести изменения в коммит


29.07.25
- git commit - создает новый коммит в текущей ветке
- git branch [name] - создает новую ветку
- git checkout [name] - переключает на другую ветку
- git checkout -b [name] - создать и переключиться на ветку
- git merge [name] - смерджить текущую ветку в ветку [name]
